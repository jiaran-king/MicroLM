**MicroLM｜中文 LLM 训练、微调与推理优化项目**

- 训练链路：搭建轻量级中文 LLM 实验框架，完成 BPE tokenizer 训练、语料编码、pretrain、SFT、LoRA 微调、KV Cache 推理 的核心链路闭环
- 模型架构：从零实现 Transformer LM（RoPE、RMSNorm、SwiGLU、causal self-attention），通过统一的 tokenizer、config 与 checkpoint 机制贯通训练与推理阶段
- 数据处理：构建面向预训练的数据处理链路，对原始 JSONL 语料执行 清洗、哈希稳定切分、EOS 边界标记、tokenization 与 metadata 留档；SFT 阶段将多轮对话渲染为 chat prompt，基于 **assistant-only masked loss** 对回答区间进行监督
- LoRA 微调：集成 LoRA 参数高效微调，支持适配器注入、权重 merge/unmerge 与独立保存，可训练参数仅占全参 Y%
- 推理优化：实现 KV Cache 增量解码，支持 prefill + decode 两阶段推理、top-p sampling 与 temperature scaling，decode 速度提升 X 倍
- 评测与扩展：建立固定 prompt 评测流程，系统对比 pretrain / SFT / LoRA 不同阶段模型的生成表现；进一步推进数据治理、性能 benchmark 与开源基座迁移实验
